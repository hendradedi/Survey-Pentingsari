# 🏞️ Dashboard Survei Kepuasan Wisatawan — Desa Wisata Pentingsari

Dashboard interaktif untuk menganalisis hasil survei kepuasan pengunjung **Desa Wisata Pentingsari**.

## ✨ Fitur Dashboard

- 📊 **Visualisasi Lengkap** — Distribusi skor, analisis per dimensi, demografi responden
- 👥 **Data Responden** — Tabel interaktif dengan filter gender & aktivitas
- 📈 **Rincian per Pertanyaan** — Skor rata-rata 18 butir pernyataan
- 🔍 **Detail per Responden** — Klik untuk lihat jawaban lengkap
- 📱 **Responsif** — Bisa diakses dari HP dan desktop
- 🎨 **Desain Soft & Elegan** — Tampilan profesional warna earth tone

## 🌐 Akses Dashboard

👉 **https://hendradedi.github.io/Survey-Pentingsari**

## 📊 Data Survei

| Item | Detail |
|------|--------|
| **Total Responden** | 106 |
| **Dimensi Analisis** | 7 (Kualitas Pelayanan, Kenyamanan, Keunikan, Nilai, Waktu, Minat Kunjung Ulang, Rekomendasi) |
| **Butir Pertanyaan** | 18 (Skala Likert 1-5) |
| **Lokasi** | Desa Wisata Pentingsari, Sleman, Yogyakarta |
| **Metode** | Kuesioner langsung kepada pengunjung |
